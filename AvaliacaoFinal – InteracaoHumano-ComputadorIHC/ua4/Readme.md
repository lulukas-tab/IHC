#  Desafio da UA4

 Este repositório contém a solução para um desafio UA4* utilizando **TypeScript**. 

##  O Desafio

A missão simulava a atuação como responsável por novas tecnologias em uma empresa que está migrando para o mercado digital. O objetivo final era criar um protótipo de sistema web de vendas para apresentar aos acionistas da organização.

O protótipo precisava cumprir três requisitos básicos:
* Criar uma classe Produto com pelo menos 5 atributos tipados.
* Criar uma classe Venda que recebe um array de produtos no construtor e possui um método para somar os valores.
* Instanciar pelo menos dois produtos, adicioná-los à venda e exibir a soma total.
