# Resolução da UA2 

Fala, pessoal! Este repositório traz a solução de um desafio muito bacana focado em Design Inclusivo e Acessibilidade. 

##  O Desafio

A missão era propor uma solução de interface para um sistema de gerenciamento de cursos internos de uma empresa. O grande detalhe? **Os usuários principais desse sistema (funcionários da área de treinamento) são deficientes visuais.
O objetivo é garantir que essa limitação não seja uma barreira e que eles consigam trabalhar com total autonomia e conforto.

##   Solução (Minha Proposta)

### a) Qual tipo de interface utilizar e por quê?
 A melhor abordagem é criar uma Interface de Voz  trabalhando em conjunto com uma Interface Gráfica Altamente Acessível (totalmente otimizada para leitores de tela). 

Por quê? 
Pessoas com deficiência visual dependem de estímulos auditivos e táteis para navegar. 
A **Interface de Voz** permite que eles interajam com o sistema, busquem cursos e cadastrem informações usando apenas comandos falados, o que agiliza muito o trabalho. 
 Já a **Interface Gráfica Acessível** (feita seguindo as diretrizes WCAG) garante que o código HTML seja limpo e semântico para que os "Leitores de Tela" consigam ler a página de forma lógica e fluida, caso o usuário prefira navegar pelo teclado.

### b) Que tipo de equipamento deverá ser utilizado?
Para que a experiência seja completa e eficiente, o setup ideal no ambiente de trabalho seria:
1. Headsets (Fone): Essenciais para que o usuário ouça o retorno do leitor de tela/interface de voz com clareza e consiga dar comandos de voz sem que o barulho do escritório atrapalhe (e com privacidade).
2. Teclado Físico (com ou sem marcações Braille): A navegação via teclado é o método número 1 de quem usa leitores de tela. Atalhos de teclado são fundamentais.
3. Softwares Leitores de Tela: Programas instalados no computador (como o NVDA, JAWS ou o próprio Narrador do Windows).
4. Display/Linha Braille (Opcional, mas recomendado): Um equipamento que se conecta ao PC e "traduz" o texto da tela em pontos Braille físicos, ajudando quem prefere a leitura tátil.
