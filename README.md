#  Avaliação Final: Ua1, UA2, UA3, UA4

Fala, pessoal! Bem-vindos ao repositório da minha Avaliação Final. 

Este espaço foi criado para documentar e entregar a resolução de três desafios práticos propostos para avaliar conhecimentos em Design de Interfaces (UX/UI), Inclusão Digital e Programação Orientada a Objetos.

Abaixo, você confere um resumo de cada projeto desenvolvido:

##  Os Desafios

### 1. Refatoração de UI: Sistema de Folha de Pagamento
* **O Problema:** Um modal de exclusão de usuário com botões ambíguos ("Avançar" e "Retornar") que causava confusão e baixa conversão.
* **A Solução:** Aplicação de heurísticas de usabilidade e melhoria no microcopy. Trocamos os textos por ações claras (**"Cancelar"** e **"Excluir Usuário"**), adicionamos hierarquia visual (botão vermelho para ação destrutiva) e um aviso de que a ação era irreversível.

### 2. Design Inclusivo: Acessibilidade para Deficientes Visuais
* **O Problema:** Projetar uma solução de locomoção externa para o "Ricardo", um usuário apaixonado por tecnologia que está perdendo a visão.
* **A Solução:** Ao invés de uma tela tradicional, a proposta focou no ecossistema que ele já usa. Idealizamos um **Aplicativo de Navegação Sensorial** que utiliza o smartphone (GPS/Sensores) como cérebro e o smartwatch como interface principal, guiando o usuário por meio de **Interface de Voz (VUI)** e **feedback háptico** (vibrações direcionais no pulso).

### 3. Protótipo Orientado a Objetos: Sistema de Vendas em TypeScript
* **O Problema:** Criar a base de um sistema web de vendas para inserir uma organização no mercado digital, modelando entidades do mundo real.
* **A Solução:** Desenvolvimento de um código em TypeScript utilizando Programação Orientada a Objetos (POO). 
    * Criamos a classe `Produto` (com atributos tipados como nome, valor, estoque).
    * Criamos a classe `Venda` (que recebe um array de produtos e possui um método interno para calcular e retornar a soma total da compra).

---

### Tecnologias e Conceitos Abordados
- **Design:** UX/UI, Microcopy, Usabilidade, Acessibilidade (WCAG), VUI (Voice User Interface).
- **Desenvolvimento:** HTML, CSS, TypeScript (Classes, Métodos, Tipagem Estática e Arrays).
