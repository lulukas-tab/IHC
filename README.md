# IHC
🌿 EcoConecta - Desenvolvimento Web Acessível
Este projeto foi desenvolvido como parte de uma Atividade Avaliativa de Desenvolvimento Web, com foco em criar uma página institucional moderna, responsiva e, acima de tudo, totalmente acessível, seguindo as diretrizes do eMAG (Modelo de Acessibilidade em Governo Eletrônico).



🚀 Sobre o Projeto
A EcoConecta é uma plataforma fictícia de consultoria em tecnologia sustentável e acessibilidade digital. O objetivo do site é demonstrar como a união entre design moderno e práticas inclusivas pode democratizar o acesso à informação na web.

🎯 Objetivos da Avaliação
•	☑HTML Semântico: Uso correto de tags como <header>, <main>, <section>, <article>, <nav> e <footer>.
•	☑Acessibilidade (eMAG): Implementação de atributos alt em imagens, aria-label em links e hierarquia de cabeçalhos (h1 a h3).
•	☑Formulários Acessíveis: Uso obrigatório da tag <label> vinculada corretamente aos inputs.
•	☑CSS Moderno: Estilização externa com foco em Flexbox para organização do layout.
•	☑Design Responsivo: Uso de @media queries e unidades relativas (rem, %, vh, vw) para adaptação em dispositivos móveis.



🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando tecnologias web fundamentais, sem a necessidade de frameworks externos, para demonstrar o domínio das bases do desenvolvimento front-end:

•	HTML5: Estruturação semântica e acessível.
•	CSS3: Estilização avançada, Flexbox e Responsividade.
•	Acessibilidade Digital: Práticas de audiodescrição e navegação por teclado.



📱 Funcionalidades
1	Menu de Navegação: Links rápidos para as seções da página, com feedback visual de foco.
2	Cards de Serviços: Layout flexível que apresenta as soluções da empresa de forma organizada.
3	Links Acessíveis: Botões "Saiba Mais" com descrições específicas para leitores de tela via aria-label.
4	Formulário de Contato: Totalmente funcional e validado, com labels acessíveis e campos estruturados.
5	Responsividade Total: A página se adapta automaticamente para celulares, tablets e desktops.



🎨 Identidade Visual
A escolha das cores foi pensada para garantir um alto contraste, essencial para a acessibilidade:
•	Roxo Escuro (#4b0082): Transmite autoridade e profissionalismo.
•	Dourado (#ffd700): Utilizado para destaques e estados de foco, garantindo visibilidade.
•	Branco e Cinza Claro: Para manter a leveza e legibilidade do conteúdo textual.



📂 Como Visualizar o Projeto
6	Clone este repositório ou baixe os arquivos.
7	Certifique-se de que os arquivos index.html e style.css estão na mesma pasta.
8	Abra o arquivo index.html em qualquer navegador moderno (Chrome, Firefox, Edge, Safari).
